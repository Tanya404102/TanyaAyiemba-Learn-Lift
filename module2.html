Week 2: Introduction to Embedded C

This module introduces the fundamental programming language for embedded systems and associated development tools.

Key Learnings:

  • Introduction to embedded systems: Basic concepts relevant to embedded programming.
  • Introduction to IDEs: Familiarity with Thonny, Arduino, and Pico C/C++ SDK.
  • Introduction to C and Embedded C: Core C programming and its adaptations for embedded systems.
  • Code Publishing - Github: Using Git and GitHub for version control of embedded code.

Projects/Exercises:

Day 1: Introduction to embedded systems, Introduction to IDEs (Thonny, Arduino, Pico C/C++ SDK).
Day 2: Introduction to C and Embedded C.
Day 3: Code Publishing - Github.
