Week 3: Introduction to ESP32 and Electrical Circuits and Components

This module delves into the specifics of the ESP32 microcontroller, its integration with electrical circuits, and common electronic components.

Key Learnings:

  • Introduction to electronic circuits and components: Basic understanding of resistors, capacitors, LEDs, etc., and how they function in a circuit.
  • Introduction to Student Kits: Familiarity with the components provided in student kits for hands-on exercises.
  • Overview of ESP32 and RPi Pico, including GPIOs: Detailed understanding of the ESP32 and Raspberry Pi Pico, focusing on their General Purpose Input/Output (GPIO) pins for interfacing with external components.
  • Introduction to the Arduino IDE and programming the ESP32: Learning how to use the Arduino IDE to write and upload code to ESP32 boards.

Projects/Exercises:

Day 1: Introduction to electronic circuits and components, Introduction to Student Kits.
Day 2: Overview of ESP32 and RPi Pico, including GPIOs.
Day 3: Introduction to the Arduino IDE and programming the ESP32.
