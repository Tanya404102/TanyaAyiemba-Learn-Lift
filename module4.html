Week 4: Introduction to Python and Raspberry Pi PICO

This module focuses on getting started with Python programming, specifically for the Raspberry Pi Pico microcontroller using MicroPython and its Arduino IDE.

Key Learnings:

  • Introduction to Python: Fundamental Python syntax, data types, control flow, and functions.
  • Installation of Thonny IDE and MicroPython: Steps to set up Thonny IDE and flash MicroPython firmware onto Raspberry Pi Pico.
  • Blink PICO LED using MicroPython: How to write and deploy a simple LED blinking program using MicroPython on Raspberry Pi Pico.
  • Blink LED using Arduino: How to achieve the same LED blinking functionality using the Arduino IDE, typically with an Arduino-compatible board.

During the lessons, TinkerCAD was used for circuit simulation and visualization before actual hardware implementation.

Code Snippet: Blink LED using MicroPython (for Raspberry Pi Pico) 


import machine
import time

# Built-in LED on Raspberry Pi Pico is typically GPIO 25
led_pin = machine.Pin(25, machine.Pin.OUT)

while True:
    led_pin.value(1)  # Turn LED on
    time.sleep(1)     # Wait for 1 second
    led_pin.value(0)  # Turn LED off
    time.sleep(1)     # Wait for 1 second

Code Snippet: Blink LED using Arduino (C++) 

// Arduino code for blinking an LED (e.g., on an Arduino Uno or ESP32/Pico configured for Arduino IDE)

const int ledPin = LED_BUILTIN; // Most Arduino boards have a built-in LED connected to a specific pin

void setup() {
  // Initialize the LED_BUILTIN pin as an output.
  pinMode(ledPin, OUTPUT);
}

// the loop function runs over and over again forever
void loop() {
  digitalWrite(ledPin, HIGH);   // turn the LED on (HIGH is the voltage level)
  delay(1000);                  // wait for a second
  digitalWrite(ledPin, LOW);    // turn the LED off by making the voltage LOW
  delay(1000);                  // wait for a second
}
