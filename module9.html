Week 9: Wireless Communication Protocols

This module introduces various wireless communication protocols essential for modern embedded systems and IoT applications.

Key Learnings:

  • ESP32 Connect to WiFi: How to establish a WiFi connection using the ESP32 microcontroller.
  • ESP32 Data Upload: Methods for uploading data from ESP32 to cloud platforms.
  • ESP32 Remote Control: Implementing remote control functionalities with ESP32.
  • ESP32 Web Server: Creating a simple web server using ESP32 for control and monitoring.
  • Simple Embedded System Project V & VI: Practical application of learned concepts through two embedded system projects.

Projects/Exercises:

Day 1: ESP32 Connect to WiFi, ESP32 Data Upload, ESP32 Remote control.
Day 2: ESP32 Web server, Simple Embedded System Project V.
Day 3: Simple Embedded System Project VI.
